# Task 2 – Auto-Wiring with Visible Arduino Pins

## Features
- Arduino Uno with visible digital pins (D2–D13)
- Default auto-wiring:
  - LED → D10
  - Push Button → D2
- Pin reassignment with conflict prevention
- Wires snap directly to board pins
- Automatic Arduino code generation

## Scope
- Task 2 only
- No logic-level simulation (Task 3)

## Tech
- HTML, CSS, JavaScript
