# Arduino Web Simulator

This repository contains a **task-wise implementation** of a minimal
web-based Arduino simulator developed for the
**FOSSEE OSHW Semester Internship screening**.

## Repository Structure

- task1- 
  Web-based interface & component handling (Task 1)

- task2-  
  Auto-wiring, visible Arduino pins, configurable pin mapping (Task 2)

- task3- 
  Logic-level simulation (Task 3 – upcoming)

## How to Run

1. Open the required task folder
2. Open `index.html` in a modern browser (Chrome recommended)
3. No installation or dependencies required

## 🌐 Live Website (GitHub Pages)

👉 **Project Homepage:**  
https://anuj5642.github.io/Arduino-Web-Simulator/

From the homepage:

- Click **Task 2** to open the Task-2 auto-wiring simulator
